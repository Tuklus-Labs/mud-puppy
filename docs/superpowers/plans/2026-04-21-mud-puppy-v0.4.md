# mud-puppy v0.4 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship v0.4 of mud-puppy — true per-layer RAM→GPU streaming, measurable training throughput wins, and a standalone Phos desktop studio UI replacing the HTML dashboard and Rich TUI.

**Architecture:** Three parallel workstreams over one repo. Python workstream rewires the trainer with a prefetch-ring layer streamer, a sequence-packing collator, and a single consolidated CPU-offload path. C++ workstream scaffolds a Phos app (`ui/`) that spawns Python training sidecars, proxies the WebSocket monitor, and exposes a typed IPC surface. Web workstream builds the React frontend that renders the monitor in the Phos webview. All three land behind the existing `--monitor` WebSocket so the Phos app is just another client — the existing browser dashboard stays functional through v0.4.

**Tech Stack:** Python 3.14, PyTorch 2.10 + ROCm 7.1, transformers/peft/trl, aiohttp WebSocket, pytest. C++17, GTK4, WebKitGTK, libphos, libcurl, nlohmann-json, tomlplusplus, spdlog, CMake. React 18, Vite, Tailwind, D3, twgl.js, Zustand, TypeScript.

**Spec:** `docs/superpowers/specs/2026-04-21-mud-puppy-v0.4-design.md`

**Conventions:**
- All commits follow Conventional Commits (`feat:`, `fix:`, `test:`, `refactor:`, `chore:`, `docs:`).
- Every code task follows TDD: write failing test → run to confirm failure → implement → run to confirm pass → commit.

### Task P3: LayerStreamer — implementation

**Files:**
- Create: `mud_puppy/stream.py`

- [ ] **Step 1: Implement LayerStreamer**

Module docstring explains: 2 CUDA streams (compute + h2d), K-slot ring of GPU-resident transformer blocks, pre-forward hook on layer N prefetches N+1, post-forward/backward hook releases the oldest slot.

Public class:

```python
class LayerStreamer:
    """Stream transformer blocks between CPU pinned RAM and GPU.

    Keeps at most `prefetch_layers` transformer blocks resident on GPU at any
    time. Uses a dedicated h2d CUDA stream for async copies and synchronises
    to the compute stream via cuda events so forward pass never stalls on
    copy when prefetch is working.

    Only transformer blocks are streamed. Embedding, final layernorm, and
    lm_head stay GPU-resident always. If LoRA adapters are attached, they
    are pinned to the streamed layers and move with them.
    """

    def __init__(self, model: nn.Module, prefetch_layers: int = 2):
        ...
```

Key invariants for implementer:

1. `_find_layers(model)` reuses the detection from old `StreamWrapper` (LLaMA `model.model.layers`, GPT-2 `transformer.h`).
2. Each block's state_dict is kept as pinned CPU copies in `self._cpu_weights: list[dict[str, Tensor]]`.
3. Ring slot = dict of empty GPU tensors matching the block's weight shapes; slot is reused for whichever layer is currently living in it.
4. `pre_forward_hook(layer_idx)` → on `h2d_stream` copy `_cpu_weights[layer_idx+1]` into next slot if not already loaded; record event; compute stream waits on it.
5. `post_forward_hook(layer_idx)` → mark oldest slot available when `layer_idx >= prefetch_layers`.
6. `pre_backward_hook(layer_idx)` → ensure block is resident again (activations flow back); if released, re-fetch synchronously (rare under ckpt strategy).
7. Embeddings / lm_head / final norm moved to `cuda` once at construction.
8. Expose `self.device`, `self.prefetch_layers`, `self._ring_slots` for tests.
9. Thread-safe for single-stream training (no multi-threaded calls into forward).

```python
"""Sequence packing for causal LM training.

Concatenates multiple short examples into a single row up to `max_seq_length`,
emitting a block-diagonal attention mask and per-segment position IDs so the
model attends only within its own segment.
"""

from typing import Any, Dict, List
import torch


class PackedCollator:
    def __init__(self, max_seq_length: int, pad_to_multiple_of: int = 8,
                 pad_token_id: int = 0):
        self.max_seq_length = max_seq_length
        self.pad_to_multiple_of = pad_to_multiple_of
        self.pad_token_id = pad_token_id

    def __call__(self, examples: List[Dict[str, Any]]) -> Dict[str, torch.Tensor]:
        # Greedy first-fit into rows of max_seq_length
        rows: List[List[Dict[str, Any]]] = []
        cur: List[Dict[str, Any]] = []
        cur_len = 0
        for ex in examples:
            n = len(ex["input_ids"])
            if n > self.max_seq_length:
                # Truncate oversize examples to fit in one row alone
                ex = {k: (v[:self.max_seq_length] if isinstance(v, list) else v)
                      for k, v in ex.items()}
                n = self.max_seq_length
            if cur_len + n > self.max_seq_length and cur:
                rows.append(cur)
                cur, cur_len = [], 0
            cur.append(ex)
            cur_len += n
        if cur:
            rows.append(cur)

        batch_size = len(rows)
        seq = self._round_up(max(sum(len(e["input_ids"]) for e in r) for r in rows),
                             self.pad_to_multiple_of)

        input_ids = torch.full((batch_size, seq), self.pad_token_id, dtype=torch.long)
        labels = torch.full((batch_size, seq), -100, dtype=torch.long)
        position_ids = torch.zeros((batch_size, seq), dtype=torch.long)
        attn = torch.zeros((batch_size, seq, seq), dtype=torch.long)

        for b, row in enumerate(rows):
            offset = 0
            for ex in row:
                n = len(ex["input_ids"])
                input_ids[b, offset:offset+n] = torch.tensor(ex["input_ids"], dtype=torch.long)
                labels[b, offset:offset+n] = torch.tensor(ex["labels"], dtype=torch.long)
                position_ids[b, offset:offset+n] = torch.arange(n, dtype=torch.long)
                attn[b, offset:offset+n, offset:offset+n] = 1
                offset += n

        return {
            "input_ids": input_ids,
            "attention_mask": attn,
            "labels": labels,
            "position_ids": position_ids,
        }

    @staticmethod
    def _round_up(n: int, multiple: int) -> int:
        return ((n + multiple - 1) // multiple) * multiple
```

"""Throughput benchmark for mud-puppy.

Runs a short training loop with varied configurations and reports
tokens/sec, steps/sec, and peak VRAM. Used to validate v0.4 changes.
"""
import argparse, json, time, subprocess, tempfile
from pathlib import Path
import torch
from transformers import AutoTokenizer
from mud_puppy.config import TrainingConfig
from mud_puppy.trainer import run_training


CONFIGS = {
    "baseline":       dict(pack_sequences=False, stream=False, compile=False),
    "packing":        dict(pack_sequences=True,  stream=False, compile=False),
    "streaming":      dict(pack_sequences=False, stream=True,  compile=False),
    "packing+stream": dict(pack_sequences=True,  stream=True,  compile=False),
}


def make_dataset(path: Path, n: int = 200):
    # Varied-length Alpaca-style JSONL
    import random
    with path.open("w") as f:
        for _ in range(n):
            length = random.choice([50, 120, 300, 800, 1400])
            msg = {"messages": [
                {"role": "user", "content": "x " * (length // 2)},
                {"role": "assistant", "content": "y " * (length // 2)},
            ]}
            f.write(json.dumps(msg) + "\n")


def main():
    ap = argparse.ArgumentParser()
    ap.add_argument("--model", default="TinyLlama/TinyLlama-1.1B-Chat-v1.0")
    ap.add_argument("--steps", type=int, default=50)
    args = ap.parse_args()

    tmp = Path(tempfile.mkdtemp(prefix="mudpuppy-bench-"))
    data = tmp / "data.jsonl"
    make_dataset(data)

    results = {}
    for name, flags in CONFIGS.items():
        cfg = TrainingConfig(
            model_name_or_path=args.model,
            dataset_path=str(data),
            output_dir=str(tmp / name),
            finetuning_method="lora",
            num_epochs=1,
            batch_size=1,
            max_seq_length=1024,
            **flags,
        )
        torch.cuda.reset_peak_memory_stats()
        t0 = time.time()
        run_training(cfg)
        dt = time.time() - t0
        results[name] = {
            "wall_seconds": dt,
            "peak_vram_gb": torch.cuda.max_memory_allocated() / 1e9,
        }

    print(json.dumps(results, indent=2))


if __name__ == "__main__":
    main()
```

int main(int argc, char** argv) {
    phos::App app("mud-puppy-studio");
    mp_studio::TrainingManager tm(app);
    mp_studio::WsBridge ws(app);
    mp_studio::HfClient hf;

    app.on_ipc("run.start", [&](auto cfg) {
        auto h = tm.start(cfg);
        ws.connect(h.run_id, h.monitor_port);
        return nlohmann::json{{"run_id", h.run_id}, {"port", h.monitor_port}};
    });
    app.on_ipc("run.stop", [&](auto args) {
        std::string id = args["run_id"];
        ws.disconnect(id);
        return nlohmann::json{{"ok", tm.stop(id)}};
    });
    app.on_ipc("run.list", [&](auto) { return tm.list(); });
    app.on_ipc("hf.search_models", [&](auto args) {
        return hf.search_models(args["query"]);
    });
    app.on_ipc("checkpoint.list", [&](auto args) {
        return mp_studio::scan_checkpoints(args["output_dir"]);
    });
    app.on_ipc("dataset.preview", [&](auto args) {
        return mp_studio::dataset_peek(args["path"], args["n"]);
    });

    return app.run();
}
```
